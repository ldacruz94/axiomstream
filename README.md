
# AxiomStream

This is an event streaming platform project that I decided to
build in Java which was inspired by Kafka.

I started this project to continue exploring distributed
systems, fault-tolerance, replication, and cloud-native
architecture using K8s.

The purpose for this project is both educational and to showcase
some comprehension of distributed systems. 

I also missed working more directly with Java, so this was a nice excuse to jump back in
the language :)
